# Spotify Rooms

A social media app centered around music. Built with sqllite, flask and react.

## To run in development mode

### Initialize venv

. ./venv/bin/activate

### Initialize env variables

- export FLASK_APP=flaskr
- export FLASK_ENV=development

### Initialize DB

flask init-db

### Run server

flask run

### Run client

cd ./flaskr/client
npm run start
